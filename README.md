# REDCap Project: Survival Analysis of Time to 5% Body Weight Reduction Among Overweight Adults

## Project Overview
This report presents the results of a longitudinal survival analysis investigating the time to achieve 5% body weight reduction among overweight adults enrolled in a behavioural weight loss programme. Data were collected using REDCap with four instruments administered at baseline and monthly follow-up visits over a six-month period.
